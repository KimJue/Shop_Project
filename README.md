# 🛒 대규모 트래픽 처리를 위한 데모 쇼핑몰 프로젝트

본 프로젝트는 고가용성 B2C 쇼핑몰의 핵심 기능을 구현하고, 대규모 트래픽 환경에서의 동시성 제어 및 성능 최적화를 실험하기 위한 백엔드 데모 프로젝트입니다.

## 🚀 프로젝트 목표
- 선착순 쿠폰 발급 시 발생하는 대규모 Write 트래픽 및 동시성 제어 해결
- 상품 및 카테고리 조회 시 대규모 Read 트래픽 최적화
- 부하 테스트 도구(JMeter/nGrinder)를 활용한 개선 전후 성능 수치 비교

## 🛠️ 기술 스택
- **Backend:** Java 17, Spring Boot 3.x, Spring Data JPA
- **Database:** MySQL, Redis
- **DevOps/Tools:** Docker

## 📌 핵심 구현 기능 (예정)
- [ ] JWT 기반 회원가입 및 로그인 API
- [ ] 상품 및 카테고리 조회 (Redis 캐싱 도입 예정)
- [ ] 선착순 쿠폰 발급 시스템 (분산 락 및 메시지 큐 도입 예정)
